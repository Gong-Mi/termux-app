## 架构对比

### Java 架构
```
TerminalEmulator.java (2,797 行)
├── 状态管理 (光标、边距、DECSET 标志)
├── 转义序列解析 (状态机)
├── 屏幕缓冲 (TerminalBuffer)
├── 行缓冲 (TerminalRow)
├── 颜色管理 (TerminalColors)
├── 样式编码 (TextStyle)
└── 会话管理 (TerminalSession)
```

### Rust 架构
```
engine.rs (1,485 行)
├── ScreenState (状态管理)
├── TerminalRow (行缓冲)
├── PurePerformHandler (vte 回调)
├── CSI/OSC/ESC 处理
└── JNI 桥接
```
